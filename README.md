Harmonious
==========

A native outliner for the Mac based on Concord, Node-Webkit, JQuery, Bootstrap
		

Building
--------

Make sure you have npm installed.

Go to the main folder and type:

```
   npm install
```

Then:

```
   cd public
   npm install
```

At this point you can run grunt in the main folder


```
   cd ..
   grunt
```
